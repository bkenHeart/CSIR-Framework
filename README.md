## ğŸ“– Introduction (ç®€ä»‹)
The **CSIR** framework is an end-to-end multimodal solution designed for automated inventory management in retail coolers. It directly generates structured inventory sequences from multi-view images without requiring bounding box annotations.

**Key Features:**
- **End-to-End:** ViT encoder + LLaMA decoder paradigm.
- **StockTokenizer:** A domain-specific tokenizer for secure and efficient generation.
- **Benchmark Dataset:** Includes 15,000 training samples and 2,000 test samples covering 107 SKUs with precise structured labels.
